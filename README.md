1. Clone this repo:

   ```bash
   git clone https://github.com/UmasriMuthuvel/DreamShop.git
   cd DreamShop
   ```

2. Configure your MySQL database inside `application.properties`:

   ```
   spring.datasource.url=jdbc:mysql://localhost:3306/dreamshop_db
   ```

   ```bash

   ```
